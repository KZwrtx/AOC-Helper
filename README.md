# AOC Helper
A helper tool for Ages of Conflict to create maps and flags faster and easier.

Current version: 1.0 (release)

## About

This tool helps create maps for Ages of Conflict using heightmaps.. You can find heightmaps on the internet or create one yourself.

Why was it created? It was created because I needed some help with making maps but couldn't find any good programs on the internet.

What problems it solves for Ages of Conflict players? It solves the problem of creating random maps or generating them from heightmaps in one click.

## Features

- Map Generation
- Flag Creation
- Custom Settings
- 2 Languages Available (RU, EN)

## Installation

`1. Download the program`
`2. Extract files`
`3. Run the EXE`

## Usage

1. Open the folder with programs
2. Run the tool you want to use
3. Generate or edit your content
4. Export it to the game by transfering files to
```
\AppData\LocalLow\JoySparkGames\Ages of Conflict\
```
